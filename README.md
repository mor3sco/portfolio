## 🚀 Executando Localmente

Clone o projeto:

```bash
git clone https://github.com/mor3sco/portfolio.git
```

Acesse a pasta:

```bash
cd portfolio
```

Abra o arquivo:

```bash
index.html
```

ou utilize uma extensão como Live Server.
